Secure AirPrint for iOS devices

Breezy allows for a secure and on-premise deployment model of Apple’s AirPrint solution. This allows users to print from iOS devices using the native print functionality on these devices and Breezy’s architecture adds the ability for printer access across the different subnets in your company’s Wi-Fi deployment.

  1. iOS device searches for AirPrint-enabled printers, sees those advertised by local Beacon.

  2. Beacon encrypts document and pushes encrypted document to internal storage.

  3. (Step 3-4) Rendering takes place on-premise as Breezy Cloud provides “traffic control”.

  4. (Step 5-7) Connector obtains rendered document from on-premise storage and sends to printer. The Breezy Cloud logs print job for compliance and cost recovery.
